🎯 拟建设计：domains/infringement/ 模块

对标 domains/inventiveness/（创造性判断），构建完整的领域 Agent 模块

原告视角 (Patentee/Plaintiff)
1. 权利基础审查
→ 专利权有效性 / 保护范围确定
2. 侵权事实举证
→ 全面覆盖比对 / 等同认定
3. 诉讼策略建议
→ 管辖选择 / 证据保全 / 临时禁令
4. 损害赔偿计算
→ 四层递进模型 / 合理开支
被告视角 (Accused/Defendant)
1. 不侵权抗辩
→ 特征缺失 / 不相同不等同
2. 法定抗辩事由
→ 现有技术 / 先用权 / 合法来源 / 权利用尽
3. 专利效力挑战
→ 无效宣告 / 诉讼中止策略
4. 赔偿抗辩
→ 专利贡献率 / 合法来源免责
