## 核心裁判规则（从案例提炼）

### 规则 1：判断主体与标准

> **来源**: invalidation_decision_018（美阿沙坦钾案）
>
> 判断说明书是否公开充分，应当**以本领域技术人员作为判断主体**，根据说明书**整体内容**结合其掌握的**普通技术知识**进行评价。

### 规则 2：功能性描述 ≠ 公开不充分

> **来源**: invalidation_decision_021（CMOS 传感器案）+ patent_exam_2012_a26_3_mechanical
>
> 仅给出功能性描述**不一定**等于公开不充分。如果该功能的实现方式是**本领域公知**的，则功能性描述是允许的。关键在于该技术手段是否属于本领域常规技术。

### 规则 3：充分公开标准因技术领域而异

> **来源**: patent_exam_2011_a26_3_chemical vs patent_exam_2012_a26_3_mechanical
>
> - **化学/医药领域**：要求更高——通常需要多个具体实施例 + 实验数据
> - **机械结构领域**：相对宽松——一个完整实施例 + 附图即可满足
> - **电子/通信领域**：如果算法步骤描述清楚，可满足 26.3

### 规则 4：公开不充分的四种经典情形

> **来源**: patent-core.yaml + 审查指南 20201
>
> 1. **缺少关键技术手段** — 未给出实施发明所需的核心技术手段
> 2. **技术手段含糊不清** — 使用了"合适的""根据需要调节"等开放性表述而无具体范围
> 3. **仅给出任务/设想** — 只提出了问题或目标，未给出具体技术方案
> 4. **实验数据不足** — 宣称的技术效果缺乏实验数据支持

## 第一部分：无效宣告决定案例

### 案例 1：玻璃板弯曲回火工作站（invalidation_decision_006）

| 属性 | 内容 |
|------|------|
| **专利号** | 95104781.7 |
| **技术领域** | 机械制造（玻璃加工设备） |
| **关键争议** | 技术特征"带气垫的板簧"导致权利要求2和3公开不充分 |
| **请求人主张** | 功能性描述导致无法确定具体结构，说明书公开不充分（26.3）；同时主张 26.4（得不到支持）和 22.3（创造性） |
| **结论** | **全部无效** |
| **指导意义** | 功能性描述（"带气垫的板簧"）如果未给出具体的结构实现方式，且该结构并非本领域常规技术，可能构成 26.3 缺陷。此案例展示了功能性描述被认定为公开不充分的边界。 |

### 案例 2：药物前体化合物美阿沙坦钾（invalidation_decision_018）★ 核心案例

| 属性 | 内容 |
|------|------|
| **专利号** | 200580013073.3 |
| **技术领域** | 化学/医药（血管紧张素II受体拮抗剂） |
| **关键争议** | 本专利说明书是否对化合物（美阿沙坦钾盐）的制备作出了充分公开 |
| **请求人主张** | 说明书不符合 26.3 的规定（具体理由涉及化合物前药制备中的公开充分性问题）；同时主张 22.3（创造性） |
| **结论** | **全部无效** |
| **决定要旨（原文）** | "判断说明书是否公开充分，应当**以本领域技术人员作为判断主体**，根据**说明书整体内容结合其掌握的普通技术知识**进行评价。" |
| **指导意义** | ★★★ 这是数据集中最关键的 26.3 案例。确立了两个核心原则：(1) 判断主体是"本领域技术人员"；(2) 评价依据是"说明书整体内容 + 普通技术知识"。化学领域专利因公开不充分被无效的典型案例。 |

### 案例 3：CMOS 影像传感器封装（invalidation_decision_021）★ 防御案例

| 属性 | 内容 |
|------|------|
| **专利号** | 201220535117.6 |
| **技术领域** | 电子/半导体（影像传感器封装） |
| **关键争议** | "透明盖板与影像感测芯片之间通过一个支撑结构保持预定间距"是否构成公开不充分 |
| **请求人主张** | 说明书仅给出功能性描述，未公开支撑结构的具体形状、材料和固定方式（26.3）；同时主张 26.4（不清楚）、22.2（新颖性）、22.3（创造性） |
| **专利权人答辩** | 支撑结构的具体实现方式属于本领域常规技术手段（蚀刻/光刻工艺形成的间隔柱是已知技术） |
| **结论** | **维持有效**（26.3 无效理由被驳回） |
| **决定要旨** | 如果本领域技术人员能够确认说明书记载的内容...（26.3 理由不成立） |
| **指导意义** | ★★★ 这是"功能性描述 ≠ 公开不充分"的权威案例。确立了关键原则：**本领域技术人员的常规知识和实验能力可以补充说明书未明确记载的内容**。但需注意：如果该技术手段是发明的关键创新点，或存在多种可能实现方式且适用条件不同，则功能性描述仍可能构成公开不充分。 |

## 第二部分：专利代理人考试真题（26.3 专题）

### 真题 1：化学催化剂（patent_exam_2011_a26_3_chemical）★

| 属性 | 内容 |
|------|------|
| **来源** | 2011 年全国专利代理人资格考试《专利代理实务》 |
| **技术领域** | 化学（石油裂化催化剂） |
| **案情** | 说明书仅记载"取原料 A 和原料 B，在高温下反应，得到催化剂产品"，但未公开：化学名称/结构式、温度范围、反应时间/压力/溶剂、活性测试方法、对比实验数据 |
| **官方结论** | **存在严重公开不充分**：缺少关键技术手段（原料不明）+ 技术手段含糊不清（"高温"无具体范围）+ 实验数据不足（未证明"提高 30%"） |
| **修改建议** | 补充 CAS 号、温度范围（如 200-400°C）、工艺参数、至少一个完整实施例及对比数据 |

**四种公开不充分情形的对应关系**：

| 情形 | 本案表现 |
|------|----------|
| ① 缺少关键技术手段 | ✗ 原料化学名称/结构式未公开 |
| ② 技术手段含糊不清 | ✗ "高温"无具体温度范围 |
| ③ 仅给出任务/设想 | — |
| ④ 实验数据不足 | ✗ 未提供活性测试数据和对比实验 |

## 使用指南：智能体如何利用本案例集

### 场景 1：判断"功能性描述是否构成公开不充分"

1. 检索案例 **invalidation_decision_021**（CMOS 传感器）→ 功能性描述被接受
2. 检索案例 **invalidation_decision_006**（玻璃板工作站）→ 功能性描述不被接受
3. **判断要点**：该功能的具体实现方式是否属于本领域公知常识？涉案专利是否提供了至少一个具体实施例？

### 场景 2：评估化学/医药领域专利的 26.3 合规性

1. 检索案例 **invalidation_decision_018** → 化学领域 26.3 标准
2. 检索真题 **patent_exam_2011_a26_3_chemical** → 四种公开不充分情形的完整分析
3. **判断要点**：是否公开了关键工艺参数？是否有实验数据支持技术效果？

### 场景 3：评估机械/结构类专利的 26.3 合规性

1. 检索真题 **patent_exam_2012_a26_3_mechanical** → 机械领域 26.3 标准（相对宽松）
2. 检索案例 **invalidation_decision_024** → 机械领域 26.3 被无效的边界案例
3. **判断要点**：是否有至少一个完整实施例？附图是否清晰展示了关键结构？

### 场景 4：分析 26.3 无效策略的成功率

- 13 件案例中：7 件无效成立（54%）、3 件被驳回（23%）、2 件部分成立（15%）、1 件结果未明
- 化学/医药领域成功率较高
- 通信/电子领域成功率与领域复杂度和说明书撰写质量高度相关
